Lumi is our project for the contest of IT projects "Perspective".

## Quickstart
1. **Clone the repository**
    ```shell
    git clone https://github.com/ss4444/kidney.git
    ```
2. **Create .env file**
    ```shell
    POSTGRES_DB=
    POSTGRES_USER=
    POSTGRES_PASSWORD=
    POSTGRES_HOST=
    POSTGRES_PORT=
    NEXT_PUBLIC_SERVER_URL=
    ```
3. **Run**
    ```
    docker-compose start
    ```
